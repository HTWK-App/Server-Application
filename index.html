Getting Started

preconditions:

  • installed jre
  • installed maven (http://books.sonatype.com/mvnref-book/reference/installation-sect-maven-install.html)
  • webbrowser (like Chrome, Firefox, ...)

Used Frameworks

  • Spring Framework for WebMVC
  • Spring Security for crypto functioniality
  • Spring Integration for feed and mail support
  • Java Image Scaling
  • Apache Commons IO for streams and writers
  • Apache Commons Lang for stringmanipulation
  • Google Guava
  • Google Gson Library
  • JSoup for html parsing and conversion
  • ...
